Welcome to Car Share IOT Application’s documentation!¶

Our team has been contacted by a car share company to develop an automatic Car Share System. This system is used to book, find and unlock and lock a car. In addition, the customer can report some issues with the car to help the company to maintain the cars. You will create an application for four types of users: customer, company manager, engineers and system administrator.

In summary, the implementation of this assignment involves the following components:

  • Python documentation toolssuch as Sphinx

  • Practice third party API

  • Unit testing in Python

  • Socket Programming

  • Writing your own API using Python’s microframework Flask

  • AI features such as facial recognition

  • Programming with Cloud databases
